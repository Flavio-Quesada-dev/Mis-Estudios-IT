# Proyectos de C++ para la UTU
# 💻 Proyectos UTU - Tecnologías de la Información

Este espacio está dedicado a mis estudios en el **Polo Educativo Tecnológico Vista Linda**. Aquí guardo ejercicios y proyectos de **C++** y lógica de programación.

## 🎯 Objetivos:
- Dominar la programación estructurada y orientada a objetos.
- Construir una base sólida en Algoritmos para el ingreso a la **FING (UdelaR)**.
- Aplicar buenas prácticas de desarrollo en sistemas de escritorio.

## 🛠️ Herramientas:
- **Lenguaje principal:** C++ (.cpp)
- **Compilador:** G++ (GNU Compiler Collection) en Linux Mint.
- **Entorno:** VS Code / Terminal de Linux.
